`preset-next` is the set of plugins that will become the [defaults](./preset-default.md) in version 6. To enable these plugins, use the
`--preset=next` option on the [command line](./command-line-options.md).

## Plugins List

- removeXMLProcInst
- removeComments
- removeMetadata
